HTB: MachineName

Difficulty: Medium • OS: Linux • Tags: Enumeration, Exploitation, PrivEsc

Enumeration

Nmap scan revealed SSH and HTTP open. Web enumeration discovered a login portal vulnerable to SQLi.

Exploitation

Used SQLMap to dump credentials. Logged in via SSH. Found a custom script running as root.

Privilege Escalation

The script was writable and executed on a timer. Replaced it with a reverse shell and caught root.
